# MINI STUDENT RESULTS PROCESSING SYSTEM 

A C# console application that allows users to enter student details and scores for 5 courses, then calculate each student's total, average, grade, and academic status. The program features a menu-driven interface with options to enter results for at least 3 students, view formatted reports showing all calculations, and exit the application. It includes score validation to ensure all marks are between 0-100, and automatically determines grades (A-F) and pass/fail status based on the average score.
